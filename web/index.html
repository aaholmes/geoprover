Three-Tier Architecture

Tier 1: Symbolic Deduction (52 forward-chaining rules)
  Input: ProofState (points, facts, goal)
  Output: saturated state (all derivable facts)
  If goal in facts -> PROVED

Tier 2: MCTS Tree Search (200 iterations, depth 3)
  At each node: generate candidate constructions
  Select via UCB/PUCT with NN priors
  Evaluate: run Tier 1 deduction, then NN value

Tier 3: Neural Oracle (GeoTransformer, ~5M params)
  Input: tokenized proof state text
  Output: policy (construction priors) + value (solve probability)
  V = sigmoid(v_logit)

Training Pipeline:
  Phase A: Synthetic pre-training (50K examples)
  Phase B: Supervised fine-tuning (JGEX solved problems)
  Phase C: Expert iteration (MCTS self-play)

Deduction Rules (52 total)

Parallel/Perp (6): transitive parallel, perp-to-parallel, corresponding angles, alternate interior, perp-parallel transfer, two equidistant perp

Congruence (7): transitive congruent, isosceles base angles, isosceles converse, perp+midpoint, SAS, ASA, SSS

Angles (4): transitive equal angle, perpendicular angles, eqangle+perp, AA similarity

Circle/Cyclic (7): circle equidistance, cyclic from oncircle, inscribed angles, inscribed converse, cyclic+parallel, cyclic+equal angle, opposite angles cyclic

Midpoint (5): midpoint definition, midpoint converse, midline parallel, midpoint diagonal, midpoint parallelogram

Collinear (3): line extension, transitivity, parallel shared point

Ratio (6): transitive, ratio=1, midpoint ratio, Thales, congruent ratio, converse Thales

Other (14): Thales theorem, equidistant midpoint, perp bisector, congruent oncircle, equal angles to parallel, quadrilateral rules (4), parallel projection, tangent rules (2), angle bisector rules (2), orthocenter concurrence
